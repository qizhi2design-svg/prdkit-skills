# 功能说明标注模板

用于描述产品功能需求和规格说明。

## 基础结构

```markdown
## 功能说明
[功能的核心价值和目标用户]

## 交互流程
1. [步骤1]
2. [步骤2]
3. [步骤3]

## 注意事项
- [注意点1]
- [注意点2]
```

## 详细结构

```markdown
## 功能说明
**目标用户**: [谁会使用这个功能]
**核心价值**: [这个功能解决什么问题]
**使用场景**: [在什么情况下使用]

## 交互流程
### 正常流程
1. [步骤1]
2. [步骤2]
3. [步骤3]

### 异常流程
- **场景1**: [异常情况] → [处理方式]
- **场景2**: [异常情况] → [处理方式]

## 数据说明
- **输入**: [需要什么数据]
- **输出**: [产生什么数据]

## 注意事项
- [注意点1]
- [注意点2]
- [边界情况说明]
```

## 使用建议

1. 明确目标用户：清楚说明谁会使用这个功能
2. 突出核心价值：解释功能解决什么问题
3. 完整的流程：包含正常流程和异常流程
4. 注意边界情况：在注意事项中说明特殊情况和边界条件
