# 产品定型参考

## 1. 商业属性

二选一：

- 商业化产品：对外销售或交付的软件产品，包含 SaaS、私有化部署、混合模式
- 企业自研系统：企业内部自用系统，不直接对外销售

## 2. 功能类型

四选一：

- 业务型管理软件：多角色协同、长链路流程、强状态流转
- 工具型软件：单一问题域、短链路、强调效率
- 交易型平台：撮合交易、支付结算、履约售后
- 基础服务型：抽象能力中心，供其他系统调用

## 3. 判断提示

### 商业化产品常见信号

- 提到客户、付费、续费、定价、版本、市场、竞品

### 企业自研系统常见信号

- 提到内部团队、组织效率、部门协同、内部审批、内部后台

### 业务型管理软件常见信号

- 提到多角色、多状态、审批、台账、配置、流程

### 工具型软件常见信号

- 提到提效、快捷操作、轻量功能、个人使用

### 交易型平台常见信号

- 提到商品、订单、支付、结算、履约、售后

### 基础服务型常见信号

- 提到开放接口、鉴权、消息、支付中心、能力中台、SDK

## 4. 章节侧重点

- 商业化产品：侧重客户价值、商业目标、增长策略
- 企业自研系统：侧重业务价值、ROI、采纳率、运营落地
- 业务型 / 交易型：侧重流程、状态、角色与权限、数据模型
- 工具型：侧重易用性、效率指标、快捷路径
- 基础服务型：侧重接口、稳定性、性能、调用链路、隔离
