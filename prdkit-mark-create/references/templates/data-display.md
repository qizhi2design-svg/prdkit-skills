# 数据展示标注模板

用于说明数据来源、展示规则和异常处理。

## 基础模板

```markdown
## 数据说明
[数据的来源和含义]

## 展示规则
[数据如何展示]

## 异常处理
[数据异常时如何处理]
```

## 详细模板

```markdown
## 数据说明
**数据来源**: [API/本地计算/缓存]
**数据类型**: [字符串/数字/日期等]
**更新频率**: [实时/定时/手动]

## 展示规则
### 正常展示
[正常情况下如何展示]

### 格式化规则
- 数字: [千分位/小数位]
- 日期: [格式]
- 文本: [截断规则]

### 排序规则
[如何排序]

## 异常处理
- **数据为空**: [显示什么]
- **数据加载失败**: [显示什么]
- **数据格式错误**: [显示什么]

## 交互说明
- 点击: [触发什么]
- 悬停: [显示什么]
```

## 示例：商品价格展示

```markdown
## 数据说明
**数据来源**: GET /api/products/:id
**数据类型**: 数字（单位：分）
**更新频率**: 实时（每次进入页面重新加载）

## 展示规则
### 正常展示
- 转换为元（除以100）
- 保留两位小数
- 添加货币符号「¥」
- 千分位分隔

示例：123456 分 → ¥1,234.56

### 格式化规则
- 原价: 灰色，删除线
- 现价: 红色，加粗，大字号
- 折扣: 橙色标签，显示「X折」

### 排序规则
- 默认按价格从低到高
- 支持切换为从高到低

## 异常处理
- **价格为0**: 显示「免费」
- **价格为空**: 显示「价格待定」
- **价格异常（负数）**: 显示「--」并上报错误
- **加载失败**: 显示「加载失败」+ 重试按钮

## 交互说明
- 点击价格: 跳转到商品详情页
- 悬停价格: 显示价格历史趋势（如果有）
```

## 使用建议

1. **明确数据来源**：说明数据从哪里来，如何获取
2. **定义展示规则**：清楚说明数据如何格式化和展示
3. **处理异常情况**：列出所有可能的异常情况及处理方式
4. **考虑交互**：说明用户可以对数据做什么操作
5. **更新频率**：说明数据多久更新一次
