# 数据展示标注模板

用于说明数据来源、展示规则和异常处理。

## 基础结构

```markdown
## 数据说明
[数据的来源和含义]

## 展示规则
[数据如何展示]

## 异常处理
[数据异常时如何处理]
```

## 详细结构

```markdown
## 数据说明
**数据来源**: [后端服务/本地计算/缓存]
**数据类型**: [文本/数字/日期等]
**更新频率**: [实时/定时/手动]

## 展示规则
### 正常展示
[正常情况下如何展示]

### 格式化规则
- 数字: [千分位/小数位]
- 日期: [格式]
- 文本: [截断规则]

### 排序规则
[如何排序]

## 异常处理
- **数据为空**: [显示什么]
- **数据加载失败**: [显示什么]
- **数据格式错误**: [显示什么]
```

## 使用建议

1. 明确数据来源：说明数据从哪里来，如何获取
2. 定义展示规则：清楚说明数据如何格式化和展示
3. 处理异常情况：列出所有可能的异常情况及处理方式
4. 更新频率：说明数据多久更新一次
