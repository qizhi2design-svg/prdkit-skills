# 标注模板说明

本目录包含三种核心标注类型的模板文件。

## 模板文件

- [prd-feature.md](prd-feature.md) - 功能说明标注模板
- [interaction.md](interaction.md) - 交互行为标注模板
- [data-display.md](data-display.md) - 数据展示标注模板

## 如何使用

1. **选择合适的模板**：根据标注类型选择对应的模板文件
2. **选择模板级别**：每个文件包含基础模板和详细模板
   - 基础模板：适合简单场景，快速创建标注
   - 详细模板：适合复杂场景，需要完整说明
3. **参考示例**：每个模板文件都包含实际示例
4. **按需调整**：模板只是参考，根据实际情况增删内容

## 使用原则

- **保持简洁**：不要为了填充模板而写无用信息
- **结构清晰**：使用标题、列表、表格等组织内容
- **可操作性**：提供具体的、可执行的信息
- **关注边界**：在注意事项中说明特殊情况和边界条件
